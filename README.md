# Signal Triage Dashboard

A simple prototype for prioritizing open-source signals based on urgency, confidence, and impact.

## Purpose
This project explores how incoming information can be structured and evaluated to support faster, more reliable decision-making in high-pressure environments.

## Key Features
- Prioritization scoring (urgency, confidence, impact)
- Filterable signals by region and priority
- Simple explainability for each ranking

## Why It Matters
In fast-moving environments, information can be noisy and misleading. This project focuses on turning raw signals into something more actionable and easier to interpret.
